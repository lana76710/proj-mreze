Repozitorijum datoteka – PRMUIS projekat
TEMA: Sistem za upravljanje datotekama

Projekat implementira klijent–server aplikaciju u C# jeziku.

Funkcionalnosti
UDP prijava klijenta
TCP komunikacija za rad sa datotekama
Citanje, izmena/kreiranje i brisanje datoteka
Zakljucavanje datoteka tokom izmene
Red cekanja zahteva
Statistika po klijentu
Struktura
Biblioteka – zajednicke klase
RepoServer – server (UDP + TCP)
RepoClient – klijent aplikacija
Tehnologije
C#
TCP / UDP soketi
.NET Framework
